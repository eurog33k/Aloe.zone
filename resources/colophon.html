Aloe.zone Colophon

Here are some of the tools and services that were used to create and host the Aloe.zone Web site.

Xojo

https://www.xojo.com
Xojo is a cross-platform development tool that can be used to create software that runs on macOS, Microsoft Windows, Linux, iOS, the Web and Raspberry Pi.

Amazon Lightsail

https://amazonlightsail.com
The site is hosted on a low-cost Amazon Lightsail virtual private server. The server is configured with 4 GB RAM, 2 vCPUs, 60 GB SSD, and runs Amazon Linux.

RapidWeaver

https://www.realmacsoftware.com
The site was designed using RapidWeaver, the best web design software for macOS.

Let's Encrypt

https://letsencrypt.org
The site's SSL certificate was issued by Let's Encrypt, a free, automated, and open Certificate Authority.

Foundation for RapidWeaver

https://foundationstacks.com
The site's design uses Foundation for RapidWeaver, an implementation of the popular Foundation front-end framework by Zurb.

Hover

https://www.hover.com
The aloe.zone domain name was registered via Hover.

Nginx

https://www.nginx.com
We're using Nginx as a reverse proxy server, so that multiple Web sites and apps can be hosted on a single virtual server.
